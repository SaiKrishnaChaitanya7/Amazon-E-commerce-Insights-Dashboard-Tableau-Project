Interactive Tableau dashboard analyzing Amazon e-commerce performance through KPIs like revenue, profit, units sold, and shipment days. Includes region, country, sales channel, cost, and product-level insights for data-driven decision making.
This project presents an end-to-end Tableau dashboard that analyzes Amazon’s e-commerce performance across multiple business dimensions. The dashboard highlights key KPIs such as total revenue, total profit, units sold, and average shipment days.
It provides detailed insights through region-wise and country-wise profit analysis, sales channel performance, yearly order trends, and cost evolution. Additional visualizations include revenue vs shipment days, priority-wise orders,
and revenue/cost/profit/order splits. The dashboard is organized into four pages—Homepage, Executive Insights, Revenue Analysis, and Item Analysis—offering a comprehensive view of business operations, customer behavior, and product performance. 
This project enables stakeholders to identify growth opportunities, optimize operations, and make data-driven decisions.
